# RPG Maze Game (C++ / ncurses)

Game **Maze berbasis terminal** menggunakan **C++** dan **ncurses**, dengan beberapa level yang harus diselesaikan pemain untuk mencapai pintu keluar.

---

## Preview
Game berjalan sepenuhnya di terminal menggunakan tampilan ASCII.

Key:
- `#` : Dinding
- `S` : Start
- `E` : Exit
- `@` : Player

---

## Fitur
- **3 Level Maze**
- Kontrol keyboard (WASD & Arrow Keys)
- Menu awal (Start / Exit)
- Sistem level selesai
- Real-time movement menggunakan `ncurses`
- UI berbasis terminal (tanpa GUI)

## Tools
- **Bahasa** : C++
- **Library** : `ncurses`
- **Platform** : Linux / macOS (WSL untuk Windows)
